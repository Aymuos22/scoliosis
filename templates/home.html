Scoliosis Classification

Project Overview

Data Augmentation for Improved Model Performance – Increased dataset size from 188 to 2000 images using augmentation techniques, enhancing model generalization.

VGGNet and CNN for Scoliosis Detection – Implemented both CNN and VGGNet models separately to analyze their effectiveness in detecting scoliosis.

Web-Based Accessibility – Hosted the model online, making scoliosis detection accessible from anywhere without requiring local installations.

Automated Medical Assistance – Provides an AI-driven approach to assist doctors in detecting scoliosis faster and more accurately.

Optimized Training Process – Used transfer learning (in the VGG model) to enhance detection accuracy with limited data.

User-Friendly Interface – Designed an intuitive interface for easy image upload and quick analysis results.

Scalability – The model can be further improved by integrating larger real-world datasets for better precision.

Potential for Future Enhancements – The project can be extended to mobile applications or integrated with hospital systems for real-time diagnosis.
